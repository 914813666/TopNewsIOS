# TopNewsIOS

` 新闻头条IOS版本(Masonry版本) `

最近项目不是特别忙，本人一直做Android研发架构的，现打算用Masonry来做IOS UI AutoLayout适配，之前项目中一直使用存代码来布局，总体来说比xlib/storyborad要好维护，但代码量太多，有点冗余的感觉!<br>
